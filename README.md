# 🎓 AttendAI System

**AI-Based Attendance System using Face Recognition**

---

## 📌 Project Overview

AttendAI is a smart attendance system that uses **face recognition technology** to automatically mark student attendance.
It eliminates manual attendance and ensures accuracy, speed, and security.

---

## 🚀 Features

* 📷 Face Recognition Attendance
* 🧑‍🎓 Student Registration (with image)
* 📊 Admin Dashboard
* 📅 Daily Attendance Tracking
* ⚡ Real-time Scanning via Webcam
* 🔐 Admin Authentication System

---

## 🛠️ Tech Stack

### Frontend

* React.js

### Backend

* Flask (Python)

* OpenCV (Face Detection & Recognition)

## ⚙️ Installation & Setup

### 🔹 1. Clone Repository

```
git clone https://github.com/Shivendrasinghthakur001/AttendAI-System.git
cd AttendAI-System
```

---

### 🔹 2. Backend Setup

```
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

👉 Backend runs on:

```
http://127.0.0.1:5000
```

---

### 🔹 3. Frontend Setup

```
cd frontend
npm install
npm start
```

👉 Frontend runs on:

```
http://localhost:3000
```

## 📸 How It Works

1. Admin registers students with face data
2. Student opens attendance page
3. System captures face via webcam
4. Face is matched with stored data
5. Attendance is marked automatically
